# AI Resume Intelligence Platform

This is a full-stack web application built using the MERN stack.

## Features (Current)
- Resume Builder (React form)
- Data sent to backend using API
- Backend built with Node.js and Express

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: (Coming Soon)

## How it works
User fills the form → Data is sent to backend → Backend processes the request

- MongoDB integration
